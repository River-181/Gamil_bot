## 2. 라벨 아키텍처 v3

**총 14개 라벨 (15개 상한 준수)**

```
@GTD (4개)
 ├─ Action     → 오늘/이번주 처리할 것
 ├─ Waiting    → 상대 답변 기다리는 것
 ├─ Reference  → 나중에 참고할 것
 └─ Read       → 읽어야 할 것

@CNU (2개)
 ├─ 학생       → 교수/학사 직접 소통
 └─ 뉴스알림   → 충남대 공지/알림

@CTX (2개)
 ├─ Travel     → 여행 예약/확인
 └─ Finance    → 금융 맥락

@AUTO (5개)
 ├─ Newsletter → 뉴스레터/SaaS업데이트/커뮤니티
 ├─ Receipt    → 결제/영수증
 ├─ Notification → 서비스 알림 (약관, 시스템)
 ├─ Social     → LinkedIn/Instagram (신규)
 └─ Promo      → 브랜드 마케팅 (신규)

@SYS (1개)
 └─ Security   → 보안/로그인 알림
```

> **라벨 변경 이력**: v2 @CTX/Project 제거 (사용 빈도 낮음, Notion으로 대체). @AUTO에 Social, Promo 추가.

## 3. 자동 필터 설계 v3

### 3-1. 기존 필터 (유지 또는 수정)

**[보안] @SYS/Security - 인박스 유지, 중요 표시**
```
조건: subject:("login" OR "OTP" OR "보안" OR "새로 로그인" OR "security" OR "인증")
     OR from:(noreply@email.apple.com)
액션: Apply @SYS/Security / Mark Important / Never Skip Inbox
```

**[CNU OTP] @SYS/Security**
```
조건: from:(cnuportal@cnu.ac.kr) subject:("OTP")
액션: Apply @SYS/Security / Skip Inbox / Mark as Read
```

**[CNU 일반] @CNU/학생 - 인박스 유지**
```
조건: from:(@cnu.ac.kr) NOT subject:("OTP" OR "공지" OR "안내" OR "뉴스")
액션: Apply @CNU/학생 / Never Skip Inbox / Star
```

## 4. 수동 처리 프로토콜

### 4-1. 매일 인박스 처리 (5분)

```
@GTD/Action   → 오늘/이번주 내가 해야 할 것
@GTD/Waiting  → 상대방 답변 기다리는 것
@GTD/Reference → 나중에 참고할 것 → Archive
삭제          → 필요 없는 것
```

### 4-2. 주간 루틴 (10분, 월요일 아침)

1. @GTD/Waiting 점검 - 답변 온 것 처리, 독촉 여부
2. @SYS/Security 확인 - 보안 이슈
3. @AUTO/Newsletter 샘플링 - 읽을 것 Raindrop 저장
4. @CTX/Finance 확인 - 이상 결제
5. @AUTO/Social 일괄 확인 - LinkedIn 초대 수락/거부

### 4-3. 월간 루틴 (20분)

1. @AUTO/Promo 일괄 삭제 (읽지 않은 것 전부)
2. 불필요 구독 제거
3. 신규 발신자 30일 패턴 → 필터 추가 여부 판단
4. 라벨 15개 상한 점검

## 5. OpenClaw AI 에이전트 연동

### 5-1. OpenClaw 역할

| 역할 | 담당 |
|------|------|
| 라벨/필터 구조 설계 및 유지 | 사람 (나) |
| 분류된 메일 읽기, 요약, 답장 초안 | OpenClaw |
| 중요 메일 메신저 푸시 알림 | OpenClaw |
| @AUTO/Promo 30일 자동 삭제 | OpenClaw heartbeat |
| @AUTO/Social 주간 요약 | OpenClaw heartbeat |
| 신규 발신자 감지 및 리포트 | OpenClaw |

### 5-2. OpenClaw 자동화 스크립트 (예시)

**@AUTO/Promo 자동 삭제 (월 1회)**
```python
# OpenClaw heartbeat task
def cleanup_promo():
    """30일 이상 된 Promo 라벨 메일 삭제"""
    cutoff = datetime.now() - timedelta(days=30)
    query = f"label:AUTO/Promo before:{cutoff.strftime('%Y/%m/%d')}"
    messages = gmail.search(query)
    for msg in messages:
        gmail.trash(msg.id)
    return f"Deleted {len(messages)} promo emails"
```

**@AUTO/Social 주간 요약 (월요일)**
```python
def weekly_social_summary():
    """LinkedIn/Instagram 주요 알림 요약해서 WhatsApp으로"""
    messages = gmail.search("label:AUTO/Social newer_than:7d")
    linkedin_invites = [m for m in messages if 'linkedin' in m.sender]
    summary = f"LinkedIn 초대: {len(linkedin_invites)}건"
    whatsapp.send(summary)
```
