# GitHub Secret Scanning / Push Protection Enablement

## Repository
- [River-181/Gamil_bot](https://github.com/River-181/Gamil_bot)

## UI Steps (Owner/Admin)
1. Open repository `Settings`
2. Open `Security` -> `Code security and analysis`
3. Turn on `Secret scanning`
4. Turn on `Push protection`

## Verification
1. `Security` tab should show secret scanning enabled
2. `Code security and analysis` should show both features as `Enabled`
3. Optional: push a fake test secret to confirm push protection blocks it
