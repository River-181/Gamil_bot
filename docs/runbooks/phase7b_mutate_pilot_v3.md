# Phase 7-b 실제 Mutate 파일럿 Runbook (v3)

## 목적
승인 기반으로 실제 Gmail mutate를 소량(5~10건)만 적용해 운영 안전성을 검증한다.

## 파일럿 고정 범위
- 대상 건수: 최소 5건, 최대 10건
- 시간창: 최근 24시간 수신
- 적용 규칙: 일반 분류 규칙만 허용
- 제외 규칙: 보안/계정 보호 계열 강제 변경 금지

## 실행 전 필수 게이트
1. `tests/plans/phase5_final_verdict_v3.md`가 `PASS`
2. `tests/plans/phase7_pilot_execution_20260224.md`(dry-run 리뷰) 완료
3. 실행 직전 백업 스냅샷 생성 및 해시 기록
4. 사용자 승인 문구 확인

## 승인 문구(고정)
"Phase 7-b 실제 mutate 파일럿(최대 10건)을 승인합니다. 이상 징후 발생 시 즉시 중단하고 롤백 절차를 수행합니다."

## 실행 규칙
1. 한 배치당 최대 10건 초과 금지
2. 적용 중 오류율 10% 초과 시 즉시 중단
3. 보안 관련 오분류 1건 발생 시 즉시 중단
4. 인박스 누락 의심 징후 발견 시 즉시 중단

## 실행 후 검증
1. 변경 대상/변경 결과(diff) 기록
2. 성공/실패/스킵 통계 기록
3. 오분류 여부 수동 검토
4. 다음 단계 판단:
- 확대 적용
- 규칙 보정 후 재파일럿
- 즉시 롤백

## 롤백 트리거
- 보안 라벨 오적용/누락
- 우선순위 충돌로 다중 분기
- 사용자 체감 장애(중요 메일 누락)
